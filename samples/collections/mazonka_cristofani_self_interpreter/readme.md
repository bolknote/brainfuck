# A Very Short Self-Interpreter

Source: https://arxiv.org/html/cs/0311032/

This collection contains BF/Dbfi-BF programs from Oleg Mazonka and Daniel B. Cristofani, "A Very Short Self-Interpreter".
Dbfi-BF uses `!` as a code/data separator, so example files include both the interpreted program and its input where appropriate.
The C++ interpreter appendix is intentionally omitted because `samples/` keeps programs and short descriptions, not implementation source.
