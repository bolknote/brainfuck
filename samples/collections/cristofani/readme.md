# Daniel B. Cristofani Brainfuck samples

Source: https://www.hevanet.com/cristofd/brainfuck/

This collection contains Daniel B. Cristofani's Brainfuck programs from the source page, including files downloaded from the site and matching files that were already present in this repository.
File names are normalized to lowercase snake_case where needed; original names are visible on the source page.

`tests_legacy.bf`, `short_legacy.bf`, and `numwarp_legacy.bf` are older local variants that differ from the current files published at the source URL.
